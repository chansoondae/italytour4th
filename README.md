# 이탈리아 아트 투어 9일 — 일정표 앱

하나투어 단독 패키지 **이탈리아 아트 투어 9일** (상품코드: EWQ111260615TW5)을 위한 모바일 전용 일정 안내 웹앱입니다.

- 여행 기간: 2026년 6월 15일(월) ~ 6월 23일(화)
- 항공: 티웨이항공 인천↔로마 직항 (TW0405 / TW0406)
- 인솔자: 임길원 작가 (010-4722-5319)

---

## 주요 기능

- **홈**: 투어 핵심 정보(항공편, 숙박, 컨셉 등), 9일 일정 카드, 포함/불포함 사항, 인솔자 연락처
- **일별 상세**: 각 날짜별 관광지·식사·이동 일정, 구글맵 연동, 유튜브 영상 플레이어
- **여행 정보**: 포함/불포함 사항, 선택관광, EES 안내, 소매치기 주의, 환전·유심·옷차림 등
- 하단 네비게이션으로 홈 / Day 1~9 / 정보 탭 빠른 이동

## 일정 요약

| 일차 | 날짜 | 주요 일정 |
|------|------|-----------|
| Day 1 | 6/15(월) | 인천 출발 → 로마 도착 |
| Day 2 | 6/16(화) | 로마 → 피렌체 / 팔라쪼 스트로치 특별전 · 우피치 미술관 · 미켈란젤로 언덕 |
| Day 3 | 6/17(수) | 베네치아 / 페기 구겐하임 미술관 · 베니스 비엔날레 (자르디니 · 아르세날레) |
| Day 4 | 6/18(목) | 돌로미티 / 친퀘 토리 · 파쏘 지아우 하이킹 |
| Day 5 | 6/19(금) | 트레비소 → 베네치아 → 파도바 / 스크로베니 예배당 |
| Day 6 | 6/20(토) | 밀라노 → 친퀘테레 → 피사 → 몬테카티니 |
| Day 7 | 6/21(일) | 몬테카티니 → 몬테풀차노 → 발도르차 → 로마 |
| Day 8 | 6/22(월) | 로마 시내 / 바티칸 박물관 · 성 베드로 대성당 · 콜로세움 · 트레비 분수 · 스페인 광장 · 판테온 등 → 로마 출발 |
| Day 9 | 6/23(화) | 인천 도착 |

## 기술 스택

- [Next.js 16](https://nextjs.org) (App Router, Turbopack)
- TypeScript
- Tailwind CSS
- shadcn/ui

## 로컬 실행

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 에서 확인할 수 있습니다.
